## Контрольная работа по модулю 3. Основные механизмы платформы. Программирование форм документов
## Тема 3.1. Программирование форм документов

Форма 1С это:

runtime-экземпляр платформенного UI-объекта,
созданный по метаданным формы,
содержащий состояние, реквизиты, элементы интерфейса,
команды и обработчики,
связанный с объектом данных,
живущий в памяти сеанса до закрытия формы.
